# Protótipo de Teste Técnico — Versão neutra

Resumo:
- Arquivo único `index.html` (HTML + CSS + JS mínimo).
- Mobile-first; responsivo para tablet/desktop.
- Acessibilidade básica: roles, tabindex, focus states.
- Performance: lazy-loading placeholders, layout leve.

Como testar localmente:
1. Baixe o `index.html`.
2. Abra em navegador OU rode um servidor local:
   - `npx http-server` (recomendado) e abra `http://localhost:8080`.
3. Substitua imagens por WebP em `thumb` / `img` conforme necessário.

Sugestão de apresentação (30–90s):
- "Projeto mobile-first, entregue como um único `index.html` para facilitar avaliação local. Priorizei performance, acessibilidade e organização do DOM — pronto para integrar assets reais e endpoints."
